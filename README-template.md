# ZeltTemplate

This project was built with the [Zelt Template](https://github.com/megazear7/zelt-stack-template)

## Setup

```sh
nvm use 22
npm install
npm run init
npm start
```

Open [localhost:3000](http://localhost:3000/)

## Develop

Hot reloading is in place on all files.
Review the prompt instructions under `.github/prompts` for coding guidelines.

```
npm start
```

## Lint & Fix

```
npm fix
```
